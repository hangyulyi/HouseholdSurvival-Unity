# Household Survival

Household Survival is a turn-based game where users play as a breadwinner living in five different countries with difficulties determined by various factors that's linked with the World Bank API.

The project is linked to a database (PostgreSQL) which stores player session data, country data, final points, and a leaderboard that's available in the hosted website.

This current project is able to be played as is, as it's already authenticated with an existing account but if users wished to create their own account, they should refer to the link below.

## Description
This is the repo that holds the Unity side of Household Survival (hosted at [https://household-survival-frontend.vercel.app/](https://household-survival-frontend.vercel.app/) )

Using Unity 6.2, this repo contains the following:
- Source files and scripts for a Unity project
- A completed WebGL build that can be embedded into a browser

The backend is currently connected to a deployed backend that uses the World Bank API to determine values and information for each playable country.
